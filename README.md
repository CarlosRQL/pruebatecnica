IMPLEMENTACION DE CRUD - FACTURAS
El directirio pruebastecnica corresponde al desarrollo realizado en PHP/JavaScript/JQuery/JSON y el archivo script db.sql correspode a la base de datos que en este caso fue MySQL.
Crear la base de datos con nombre prueba_factura, user: admin pass: admin
Instalar apache o nginx para poder ejecutarlo localmente (Validar puertos asignados) 
Agregar y descomprimir la librearia dompdf para poder ejecutar los pdf.
